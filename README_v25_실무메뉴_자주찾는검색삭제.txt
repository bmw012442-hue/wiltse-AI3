# ICU AI v24 - 자주 찾는 검색 메뉴 삭제 + 표/그림 표시 지원

변경사항:
1. 실무 메뉴 항목에서 '자주 찾는 검색' 삭제
2. 카드 상세창에서 표 표시 기능 추가
3. 카드 상세창에서 그림/이미지 표시 기능 추가
4. 주요 카드에 표 데이터 추가
   - Lab bottle 종류와 채혈 순서
   - BST Sliding
   - DKA protocol
   - CRRT Return
   - 수혈 절차

중요:
- 기존 DB 원본에 들어있는 표/그림 파일 자체가 아직 이미지 파일로 분리되어 있지 않으면 자동으로 그림이 생기지는 않습니다.
- 이번 버전은 카드 DB에 images 필드가 있으면 검색 결과 카드 상세창에 그림이 뜨도록 기능을 추가한 버전입니다.
- 실제 원본 표/그림 파일을 보내주면 다음 버전에서 해당 이미지들을 public/images 폴더에 넣고 카드와 연결할 수 있습니다.

GitHub 적용:
루트 public 폴더에 ZIP 안의 public 폴더 파일 전체를 덮어쓰기 하세요.
