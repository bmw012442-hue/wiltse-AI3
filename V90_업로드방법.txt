# V90 구조정리 배포 방법

## Render 배포
1. 이 zip을 풀어 GitHub 저장소에 업로드합니다.
2. 루트에 package.json, server.js, render.yaml이 있어야 합니다.
3. public 폴더 전체가 그대로 있어야 합니다.
4. Render는 npm install 후 npm start로 실행됩니다.

## GitHub Pages로만 열 때
- 루트 index.html은 public/index.html로 자동 이동합니다.
- 단, 로그인/AI 서버 기능은 Render 같은 Node 서버가 있어야 작동합니다.
