# ICU AI v22 - 홈화면 앱 이름 변경

변경사항:
- 홈화면에 추가했을 때 앱 이름을 'ICU 업무메뉴얼'로 변경
- manifest name/short_name 변경
- iPhone apple-mobile-web-app-title 변경
- 기존 V21 기능 유지

GitHub 적용:
루트 public 폴더에 ZIP 안의 public 폴더 파일 전체를 덮어쓰기 하세요.

주의:
이미 홈화면에 추가한 기존 앱은 이름이 자동으로 바뀌지 않을 수 있습니다.
기존 홈화면 아이콘을 삭제한 뒤 다시 '홈 화면에 추가'하면 새 이름으로 표시됩니다.
