# ICU AI v20 - 앱 아이콘 적용 + 설치형 웹앱(PWA)

변경사항:
- 사용자가 제공한 원형 윌스 아이콘을 앱 아이콘으로 적용
- iPhone 홈 화면 추가 아이콘 적용
- Android 설치형 PWA 아이콘 적용
- manifest.json, apple-touch-icon, favicon 추가
- service worker 추가로 앱처럼 설치 가능
- 기존 V19 기능 유지
