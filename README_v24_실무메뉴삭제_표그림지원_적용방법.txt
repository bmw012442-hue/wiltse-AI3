# ICU AI v23 - 홈화면 아이콘 변경

변경사항:
- 홈화면에 추가했을 때 보이는 앱 아이콘을 새 원형 윌스 로고로 변경
- Android/PWA 아이콘(icon-192, icon-512) 변경
- iPhone 홈화면 아이콘(apple-touch-icon) 변경
- 파비콘(favicon-16, favicon-32)도 동일 로고로 변경

적용 방법:
1. GitHub의 현재 사용 중인 public 폴더로 들어갑니다.
2. 이 ZIP 안의 public 폴더 파일들을 기존 파일 위에 덮어쓰기 업로드합니다.
3. 커밋합니다.
4. Render가 자동 배포되면 사이트를 새로고침합니다.

중요:
- 이미 홈화면에 추가한 기존 앱은 예전 아이콘이 남아 있을 수 있습니다.
- 기존 홈화면 아이콘을 삭제한 뒤, 브라우저 캐시를 새로고침하고 다시 홈 화면에 추가해야 새 아이콘이 보입니다.
- iPhone/Safari는 아이콘 캐시가 강해서, 사파리를 완전히 종료 후 다시 추가하면 더 잘 반영됩니다.
